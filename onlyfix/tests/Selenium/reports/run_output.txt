=== MANUÁLIS FUTTATÁSI UTASÍTÁSOK ===
1. pip3 install selenium pytest pytest-html
2. brew install --cask google-chrome  (ha nincs Chrome)
3. php artisan migrate:fresh --seed   (adatbázis reset)
4. Indítsd el az alkalmazást (php artisan serve / Docker)
5. cd onlyfix && python3 -m pytest tests/Selenium/ -v --html=tests/Selenium/reports/report.html
